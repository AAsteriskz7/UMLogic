# UMLogic | Learn UML Diagramming

Visit the live application: **[umlogic.vercel.app](https://umlogic.vercel.app)** 

## Overview
UMLogic is a premium, interactive learning platform designed for students and professionals mastering **Software Objects and Design**. It bridges the gap between theoretical software architecture and practical implementation through multi-modal learning modules.

![UMLogic Dashboard](/hero-viz.png)

## Key Features
- **Multi-modal Learning**: Information, construction, and quiz modes for every diagram.
- **5 Core Diagrams**: Unified support for UCD, DMD, SSD, SD, and DCD.
- **Progress Tracking**: Real-time persistence of your learning journey.
- **Case Study Driven**: Realistic scenarios based on the *CampusConnect* management system.
- **Premium UI**: Dark-mode ready, smooth animations, and interactive nodes.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS 4, Shadcn/UI
- **Animations**: Framer Motion
- **Diagramming**: Mermaid.js, Monaco Editor

## Getting Started

### Prerequisites
- Node.js 18.x or later
- npm or yarn

### Installation
```bash
npm install
```

### Development
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🤝 How to Contribute (Team)
Team members can contribute directly via the GitHub web interface:
1. **Documentation**: Expand the templates in the `docs/` folder (`ARCHITECTURE.md`, `USER_GUIDE.md`, `API_SIMULATION.md`).
2. **Logic Refinement**: "Improve" the simulated backend hooks in `lib/hooks/`. These files follow standard React patterns and are isolated from the core UI.

## 📚 Project Context
Developed to make the transition from "System Logic" to "Software Design" intuitive and visually engaging for students and early-career developers.
