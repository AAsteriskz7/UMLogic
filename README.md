## Overview
UMLogic is a premium, interactive learning platform designed for students and professionals mastering **Software Objects and Design**. It bridges the gap between theoretical software architecture and practical implementation through multi-modal learning modules.

![UMLogic Dashboard](/public/hero-viz.png)

## 🚀 Key Features

- **Multi-Modal Learning**: Every diagram type includes a detailed "Information" brief, a "Sandbox" builder, and an "Interactive Quiz".
- **5 Supported Diagram Types**:
  - **UCD**: Use Case Diagrams
  - **DMD**: Domain Model Diagrams
  - **SSD**: System Sequence Diagrams
  - **SD**: Sequence Diagrams
  - **DCD**: Design Class Diagrams
- **Smart Progress Tracking**: Real-time tracking of your mastery across all modules, including a comprehensive Final Exam simulation.
- **CampusConnect Case Study**: A consistent, GT-themed scenario (Managing student organizations) used throughout the app to demonstrate how different diagrams interact.
- **Premium UI**: Built with a sleek Georgia Tech navy/gold aesthetic, featuring smooth micro-animations and custom 3D visualizations.

## 🛠 Tech Stack

- **Framework**: [Next.js 16.1.6 (App Router)](https://nextjs.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/) & [shadcn/ui](https://ui.shadcn.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Diagram Rendering**: [Mermaid.js](https://mermaid.js.org/)
- **State Management**: Local Storage Persistence for Progress Tracking

## 🤝 How to Contribute (Team)
Team members can contribute directly via the GitHub web interface:
1. **Documentation**: Expand the templates in the `docs/` folder (`ARCHITECTURE.md`, `USER_GUIDE.md`, `API_SIMULATION.md`).
2. **Logic Refinement**: "Improve" the simulated backend hooks in `lib/hooks/`. These files follow standard React patterns and are isolated from the core UI.
